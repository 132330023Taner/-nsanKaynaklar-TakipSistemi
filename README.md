# Personel Takip Sistemi 

Bu proje, Windows Forms kullanılarak geliştirilmiş bir **Personel Takip Sistemi** uygulamasıdır.  
Proje kapsamında personel, departman, izin, maaş ve performans işlemleri yönetilmektedir.

## Kullanılan Teknolojiler
- C# (.NET Framework / WinForms)
- Katmanlı Mimari (UI, BLL, DAL)
- SQL Server / MySQL
- ADO.NET

## Uygulama Özellikleri
- Personel ekleme, silme, güncelleme
- Departman yönetimi
- İzin takip sistemi
- Maaş ve performans takibi
- Kullanıcı girişi ve yetkilendirme
- Tema (renk/font) yönetimi

## Proje Yapısı
- **personelTakip.UI** : Kullanıcı arayüzü (Windows Forms)
- **personelTakip.BLL** : İş kuralları
- **personelTakip.DAL** : Veritabanı işlemleri
- **personelTakip.Entities** : Varlık sınıfları

## Kurulum
1. Projeyi Visual Studio 2022 ile açınız.
2. Veritabanı bağlantı ayarlarını düzenleyiniz.
3. Gerekli tabloları oluşturunuz.
4. Uygulamayı çalıştırınız.
